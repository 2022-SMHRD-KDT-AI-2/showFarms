# showFarms
2nd Team Project, 학습된 YOLO 모델을 이용한 스트리밍 서비스가 탑재된 농산물 직거래 플랫폼

# 프로젝트 상세설명
1. 농산물 이미지 및 라벨링 데이터를 이용하여 YOLO 모델을 학습하여 농산물을 분류하는 '품질분류모델'을 만든다.
